Для того чтобы запустить проект нужно:

1)Скачать проект с гитхаба с помощью phpstorm, git или github desktop

1.1)Можно скопировать ссылку HTTPS и ввести в git, или через phpstorm через clone repository, либо через github desktop

1.2)В открывшимся окно вставить ссылку на репозиторий https://github.com/RomanP005/Tproject.git

2)На проект установить композер с помощью команды "composer install", и еще нужно чтобы php был глобально установлен. 

2.1)Смотите куда устанавливаетее composer, нужно чтобы он установился в ваш проект.

2.2)Обычно, если вы устанавливаете проект с помощью phpstorm путь будет такой -> "C:\Users\имя_пользователя\PhpstormProjects\название_проекта"

3)С гугл диска(https://drive.google.com/file/d/1DQpc3DWptpYhOWtjbO0j-3yzyMDCktvm/view?usp=sharing) нужно скаачать архив с названием phpmyadmin, её требуется распоковать на какой нибудь диск и запустить Ospanel.

4)Запустив OSPanel в командной строке нужно написать php artisan migrate, чтобы создалась база данных

5)Чтобы запустить сайт можно прописать php artisan serve и перейти по ссылке которая выдала командная строка

6)Так же по желанию можно просписать php artisan DB:seed для заполнения таблиц проектов и задач в базе данных

7)После того как вы зайдете на сайт нужно зарегистрироваться, желательно сразу двух пользователей чтобы увидеть весь доступный функционал.

8)Затем через одного пользователя вы создаете проект, а после при создании задачи можно задействовать второго пользователя в качестве исполнителя.

9)Остается проверить имеющийся функционал, и казнить нас за наш грязный код.
